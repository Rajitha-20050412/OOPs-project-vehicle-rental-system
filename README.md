# 🚗 Vehicle Rental Management System (Java OOPS)

## 📌 Overview
A console-based Vehicle Rental Management System built using Core Java.

## 🚀 OOPS Concepts Implemented
- Abstraction
- Interface
- Inheritance
- Polymorphism
- Encapsulation
- Composition
- Custom Exception Handling

## 🛠 Technologies Used
- **Java**
- **JDBC**
- **MySQL**
- **Collections Framework**
- **Object-Oriented Programming**
## Database Setup

1. Install MySQL
2. Open MySQL Workbench
3. Run the file:

database.sql

This will create:

- vehicle_rental database
- vehicles table
- rentals table

## Technologies Used

- Java
- OOP Concepts
- JDBC
- MySQL

javac src/Main.java  
java -cp src Main
