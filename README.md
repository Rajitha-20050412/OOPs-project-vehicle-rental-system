# 🚗 Vehicle Rental Management System (Java OOPS)

## 📌 Overview
A console-based Vehicle Rental Management System built using Core Java.

## 🚀 OOPS Concepts Implemented
- Abstraction
- Interface
- Inheritance
- Polymorphism
- Encapsulation
- Composition
- Custom Exception Handling

## 🛠 Technologies Used
- Java
- Collections Framework

## ▶ How to Run

javac src/Main.java  
java -cp src Main
